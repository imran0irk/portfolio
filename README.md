# Portfolio — Md Imran Khan

This repository contains my personal portfolio website (a simple, responsive one-page site).

## Live site
If you’re using GitHub Pages, it will usually be available at:
`https://imran0irk.github.io/portfolio/`


## Project structure
- `index.html` — main website page
- `assets/css/` — stylesheets
- `assets/js/` — JavaScript files
- `images/` — site images (profile + project images)
